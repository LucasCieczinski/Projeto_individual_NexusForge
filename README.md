# Projeto_individual_NexusForge

Repositorio voltado para o desenvolvimento do meu Projeto individual
Tema: League Of Legends

Objetivo:
Transformar dados complexos dos jogadores em insights práticos e personalizados sobre o desempenho deles nas ultimas 20 partidas, ajudando a identificar pontos fortes, áreas de melhoria e padrões de comportamento durante as partidas. Traduzindo de maneira simples e visual as estatísticas em informações estratégicas que apoiam a evolução contínua do jogador, independente do nivel ou Elo
